List of changes

  • In our initial design from P2 our pages varried in width, this has been changed to make the pages look more uniform and aesthetically pleasing. We achieved this by adding a main element with a set width.
  • We changed some filenames from P2:
    • about-us.html is now called about.html
    • contact-us.html is now called contact.html
    • order-form.html is now called order.html
    • order.html is now called ordernow.html
  • The boxes on the contact-us page no longer have a set width and height in pixles. The width of the box is deterimined by the main element width and the height is determined by the svg height and padding.
  • Added a dessert and drinks section to our menu page.
  • We didn't use JavaScript to implement the map functionality, hence there is no map.js file.

Minimum Requirements

  • Automatic element showing open/closed in header
    • Malik & Brage M finished this task
      • This component in the header will check the current time to see whether it should display as an open or closed variant using JavaScript. We will use JavaScript's Date object.

  • Nav button will update styling based on which site you are on
    • Maksymilian finished this task
      • This button will update its styling based on whether the current url path matches its target. We will use a CSS class for this.

  • An interactive map showing Papa's location
    • Brage finished this task
      • We will of course not make this map ourselves, and use tools from Google Maps to achieve this

  • Menu list implemented with pre-made JSON data and a for loop in Javascript
    • Philip finished this task
      • We will use a for loop iterating over the data in our pre-mde JSON file, which we will import using JavaScripts import function
