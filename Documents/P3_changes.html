Minimum Requirements

  • Automatically element showing open/closed in header
    • Malik will take care of this
    • This component in the header will check the current time to see whether it should display as an open or closed variant using JavaScript. We will use JavaScript's Date object.
  • Nav button will update styling based on which site you are on
    • Maximilian will take care of this
    • This button will update its styling based on whether the current url path matches its target. We will use a CSS class for this.
  • An interactive map showing Papa's location
    • Brage will take care of this
    • We will of course not make this map ourselves, and use tools from Google Maps to achieve this
  • Menu list implemented with pre-made JSON data and a for loop in Javascript
    • Philip will take care of this
    • We will use a for loop iterating over the data in our pre-mde JSON file, which we will import using JavaScripts import function
